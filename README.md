# PMKT · 存量经营分省作战 · 在线作战系统

面向中国移动 31 省移动网络**存量用户精细化经营**（离网/降档/降套/升套）的分档评估与开方作战系统。单文件（`index.html`），双击即开，无需联网。

**在线访问**：启用 GitHub Pages 后见 https://lucassu2012.github.io/PMKT/

## 功能

- **作战地图**：真·中国省级地图，按 梯队 / 加权分 / 置信度 / 华为格局 着色，点省份看详情与作战建议。
- **判级速查向导**：每维度 3 问二选一自动推 L 级，实时算加权分、定梯队（数据准备度 <L3 自动封顶 T3）。
- **双轴矩阵**：梯队 × 华为格局 打单优先级矩阵。
- **作战建议（开方规则引擎）**：四维 L 级 + 格局 + 梯队 → 方案组合 + 依赖链 + 方案③模式A/B + 方案④三件套 + 作战剧本。
- **AI 作战剧本文案**：从结构化开方生成自然语言剧本，AI草稿→机关审核→已审核 人工审闭环（生产环境接大模型）。
- **数据对接**：源系统指标按映射规则自动推 L 级，采纳差异项进机关审核队列。
- **情报校准闭环**：低置信格 → 派任务 → 代表处回填 → 机关审核 → 自动重算。
- **角色权限**：机关（全权）/ 代表处（本省回填、格局轴隐藏）/ 领导（只读）。
- **导出**：内部完整版 / 对外裁剪版（摘除格局轴与省份排名）。

## 评估方法

四维加权（数据准备度 30% / 场景化模型 20% / 营销闭环 30% / 体验经营 20%），加权分定四梯队（T1≥4.0 / T2 3.0–3.9 / T3 2.0–2.9 / T4<2.0），数据准备度 <L3 封顶 T3。

## 技术

纯静态单文件 HTML + 原生 JS，浏览器 localStorage 持久化；中国地图为 DataV GeoJSON 投影简化后内联的 SVG。数据可在页面内「重置为初始种子」。
