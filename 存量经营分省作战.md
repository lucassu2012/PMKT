# 存量经营分省作战 · 总览

**一句话定位**：面向中国移动 31 省移动网络**存量用户精细化经营**（离网/降档/降套/升套）的**分档评估与开方作战体系**——四维 L1-L5 成熟度评估 + 梯队×格局双轴矩阵 + 诊断开方方案映射，是华为内部作战地图，可裁剪为对中国移动集团的咨询式汇报。

## 方法论主线

```
四维评估（数据30% / 模型20% / 营销闭环30% / 体验20%）
   → 加权分定四梯队（T1先锋≥4.0 / T2发展 / T3基础 / T4起步，数据<L3封顶T3）
   → 叠加华为格局轴（强/部分/友商，仅内部版）成双轴矩阵
   → 按短板维度诊断开方（四大方案映射 + 依赖链）
   → 分梯队作战（T1样板深化 / T2补短板 / T3地基+试点 / T4地基工程）
```

## 何时读本目录

- **给某省定打法/写汇报**：先查 [[02_战略规划/存量经营分省作战/分档地图-31省双轴矩阵]] 找到该省梯队×格局定位，再按 [[02_战略规划/存量经营分省作战/开方逻辑-四大方案映射]] 开方。
- **与省公司/集团客户做存量经营交流**：取 [[02_战略规划/存量经营分省作战/评估标准-四维L1-L5]] 做成熟度自评框架（对外版，见下方裁剪规则）。
- **代表处情报校准**：查分档地图中标注「低置信」的格子，即该省待验证情报清单，回填后升级置信度。
- **机关做区域打单优先级排序/资源摆布**：读双轴矩阵的打单优先级象限 + 各梯队作战模式（T1 样板深化、T2 补短板放大、T3 地基+试点、T4 地基工程）。

## 使用纪律

| 纪律 | 规则 |
| --- | --- |
| 内外版本裁剪 | 内部版含**格局轴**（华为强/部分/友商格局）与打单优先级；对外（含对中国移动集团汇报）**整体摘除格局轴与省份排名**，只保留四维评估框架、梯队分布统计与开方逻辑 |
| 置信度三级 | **高**=项目实证（知识库内有该省项目数据）；**中**=间接口径（KB 提及/集采格局等间接信息）；**低**=推断（无 KB 证据，凭公开常识判断）。低置信格子=代表处情报校准任务，不得直接对客引用 |
| 数字出处 | 本目录所有评级与数字须能回溯知识库出处；无出处的判断在卡内显式标注「推断」 |
| 刷新机制 | 分档地图与省档案随新项目实证/代表处回填**滚动刷新**：新证据到达→更新该省 L 级与置信度→重算加权分与梯队；评估标准（维度/权重/门槛）为框架共识，调整须重走机关评审 |
